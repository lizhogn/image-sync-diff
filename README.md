# Image Sync Diff

Synchronized image comparison with overlay support for VS Code.

![Extension Demo](ExtensionDemo.gif)

## Features

- **Synchronized Zoom & Pan**: Inspect details of two images simultaneously. Zooming or panning one image automatically adjusts the other.
- **Comparison Modes**:
    - **Side by Side**: View images next to each other.
    - **Dissolve**: Overlay images and adjust opacity to spot subtle differences.
- **Difference Highlighting**: Toggle a "Differences" view to visualize pixel-level discrepancies between the two images.
- **Flexible Selection**: Compare images directly from the file explorer.

## Usage

### Method 1: Compare Two Images
1. Select two images in the VS Code File Explorer (hold `Ctrl` or `Cmd` to select multiple).
2. Right-click and choose **Compare Images with Sync Diff**.

### Method 2: Compare with First Image
1. Right-click on a single image in the File Explorer and select **Set as First Image**.
2. Right-click on a second image and select **Compare with First Image**.

### Method 3: Activity Bar
1. Open the **Image Sync Diff** view in the Activity Bar.
2. You can manage the "First Image" selection from here.
