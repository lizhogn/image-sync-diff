# Image Sync Diff

A VS Code extension for synchronized image comparison.

## Features

- **Split View**: Drag and drop two images to compare them side-by-side.
- **Synchronized Zoom & Pan**: Zooming or panning one image affects the other instantly.
- **Overlay Mode**: Click the arrow button to overlay the right image onto the left for pixel-perfect comparison.
- **Zoom Indicator**: Real-time zoom level display.

## Usage

1. Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`).
2. Run `Image Sync Diff: Start`.
3. Drag and drop images into the view.
